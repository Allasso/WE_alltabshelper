You have been updated to All Tabs Helper (WE) version 1.0.6
NOTE: If this update has migrated you to your first WebExtensions version of All Tabs Helper, please observe the following:
All Tabs Helper can be accessed two ways as before, though a little differently:
Drop-down: Rather than clicking on the "All tabs" button as before, you will now need to click on the All Tabs Helper toolbar button (The "A" icon) for the menu to appear.
Sidebar: Select All Tabs Helper in View > Sidebar.
Please also read http://kevinallasso.org/alltabshelper/all_tabs_helper_webextension_info.html for a more detailed description of the WebExtensions migration.
